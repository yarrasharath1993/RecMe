# GOVERNED PROMPT: Editorial Content Outline Generator

## PURPOSE
Generate structured outlines for editorial content that:
- Separates fact from opinion
- Labels all speculative content
- Never presents opinion as fact
- Provides clear attribution

## CONSTRAINTS (MANDATORY)

### Content Classification
All generated content must be classified as:
- **FACT**: Verified from 2+ sources, can be stated directly
- **EDITORIAL**: Opinion/interpretation, must be labeled "Editorial view:"
- **SPECULATIVE**: Unverified claims, must be labeled "[Speculative]"

### Allowed Sources for FACT
- verified_fact content_type only
- trust_score >= 70
- confidence_tier: high

### Allowed Sources for EDITORIAL
- editorial content_type
- trust_score >= 50
- Must be clearly labeled

### NEVER Include
- speculative content_type without [Speculative] label
- disputed data (is_disputed=true)
- content with trust_score < 50
- unverified family claims
- fan theories as facts

## INPUT SCHEMA

```json
{
  "topic": "string (e.g., 'Career Analysis of [Person]')",
  "entity_data": { ... entity profile data ... },
  "editorial_views": {
```

## SECTION TEMPLATES

### Career Analysis Outline
```
1. Introduction (FACT)
   - Career span
   - Total filmography count
   - Primary roles

2. Career Phases (FACT + EDITORIAL)
   - Phase 1: [Name] ([Years])
   - Phase 2: [Name] ([Years])
   - *Editorial view:* [Analysis of evolution]

3. Notable Achievements (FACT)
   - Awards
   - Records
   - Milestones

4. Industry Impact (EDITORIAL)
   - *Editorial view:* [Impact analysis]
   - Supporting evidence

5. Legacy (EDITORIAL)
   - *Editorial view:* [Legacy assessment]
```

### Film Analysis Outline
```
1. Film Overview (FACT)
   - Release details
   - Cast & crew
   - Genre classification

2. Critical Reception (FACT)
   - Ratings
   - Awards

3. Cultural Impact (EDITORIAL)
   - *Editorial view:* [Impact analysis]

4. Comparative Analysis (EDITORIAL)
   - Similar films
   - Genre context
```

## PROHIBITED ACTIONS

1. DO NOT mix FACT and EDITORIAL without clear labels
2. DO NOT include SPECULATIVE content without warning banner
3. DO NOT use is_disputed data in FACT sections
4. DO NOT generate content for entities with trust_score < 50
5. DO NOT attribute opinions to the subject
6. DO NOT claim awards/achievements without verification

## QUALITY CHECKS

Before generating, verify:
- [ ] All FACT statements are from verified sources
- [ ] All EDITORIAL sections are labeled
- [ ] All SPECULATIVE content has warning banner
- [ ] No disputed data included
- [ ] Trust score meets threshold
- [ ] Sources are attributed
